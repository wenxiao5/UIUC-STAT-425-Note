In this example we are going to focus on three variables: ctn (i.e.bike share counts) as the response, and t1 real temperature and is_weekend (weekend/weekday) as predictors.

The is_weekend variable is a categorical variable that takes 0 if the day is Saturday/Sunday and 1 if it is a weekday.

So, we start by restricting the data set for our purposes (this is just done for illustration purposes):

# We keep columns 2,3,9 and sample only 50 (otherwise the plots are not easy to interpret)
seed = 5434;
bikeshares.ancova = sample_n(bikeshares[ ,c(2,3,9)], 50);
head(bikeshares.ancova);

Fit the Full Model

We start our analysis by fitting the full model, that is a model that contains both predictors (t1, is_weekend), as well as their interaction (t1:is_weekend). Essentially, the full model is a model with different intercepts and different slopes for each of the groups.

bikeshares.full = lm(cnt~ t1 + is_weekend + t1:is_weekend, data=bikeshares.ancova)
# same as lm(cnt~ t1 * is_weekend, data=bikeshares.ancova)
summary(bikeshares.full)

Fit the Additive Model

Based on its \(p\)-value, the interaction term is not statistically significant, so we fit a smaller model without the interaction term; the (additive) model. Since there are no interactions, we have two parallel lines: different intercepts and same slope.
